# QUICK START - RUN IN 3 STEPS

## Step 1: Install Dependencies
Open terminal/command prompt in the scam_detector folder and run:
```
pip install -r requirements.txt
```

## Step 2: Start the App
```
python app.py
```

You should see:
```
* Running on http://127.0.0.1:5000
* Press CTRL+C to quit
```

## Step 3: Open in Browser
Go to: http://localhost:5000

## Presentation Tips

1. **Show both extremes**: One obvious scam, one legitimate job
2. **Point out flagged phrases**: Highlight what the system detected
3. **Explain the score**: Show how each category contributes
4. **Mention limitations**: "Rule-based, works for common patterns"
5. **Talk about extensions**: "Could add ML, company verification, browser plugin"

## If Something Goes Wrong

**Port 5000 in use?**
Edit app.py, change this line:
```
app.run(debug=True, port=5000)
```
To:
```
app.run(debug=True, port=5001)
```

**Dependencies error?**
```
pip install --upgrade pip
pip install -r requirements.txt
```

**Pages look blank?**
Check browser console (F12 → Console tab) for errors
